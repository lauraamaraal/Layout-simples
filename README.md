# ✨ Minhas Viagens dos Sonhos ✨

Um layout simples e bonito para listar seus destinos de viagem dos sonhos, criado com HTML e CSS.

## 📋 Sobre o Projeto

Este é um projeto pessoal de uma **lista de desejos de viagem**, onde compartilho os 6 lugares que mais quero conhecer no mundo. Cada destino é apresentado em um card com:

- 📍 Nome e localização
- 🖼️ Imagem representativa
- 📝 Descrição personalizada
- 🏷️ Tags com características marcantes

## 🎨 Tecnologias Utilizadas

- **HTML5** - Estrutura da página
- **CSS3** - Estilização, animações e responsividade
- **Unsplash** - Imagens gratuitas e de alta qualidade

## ✨ Funcionalidades

- 📱 **Totalmente responsivo** - se adapta a qualquer tela (celular, tablet, desktop)
- 🖱️ **Efeitos hover** - cards com animações suaves ao passar o mouse
- 🎨 **Design aconchegante** - paleta de cores verde e dourado
- 🏷️ **Tags personalizadas** - cada destino tem características únicas
- 💫 **Sem JavaScript** - apenas HTML e CSS
